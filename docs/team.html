2. Pipeline flow

From ticker to report

3. Quality rules

Nothing ships unchecked

The visual style is warm, but the workflow remains deliberately skeptical: public sources, no private tokens, and explicit validation before delivery.

Default answer is no

Missing evidence is a verdict input, not a bullish placeholder. Borderline claims stay marked as insufficient until proven.

Public sources only

Filings, investor relations pages, exchange pages, reputable news, public market data, and web search.

Separate owners

Researcher, council, Bruce, Checker, Reporter, and Librarian write distinct artifacts and avoid overwriting one another.

Checker can block

The final report waits until validation is clear or the warnings are explicit, visible, and accepted by the workflow.
